 # 🏭 工业视觉通信与控制上位机系统 (Vision-Control-Hub)
> 
> ### 📖 项目简介
> 本项目是一个基于 **C# WPF** 开发的工业级机器视觉上位机系统。成功深度集成了 **海康 VisionMaster 4.4.0 SDK**，并实现了视觉识别与底层硬件（PLC/机械臂）通信的全自动化闭环。
> 
> ### 🚀 核心亮点 (Features)
> *   **双图实时渲染**：嵌套 `VmRenderControl` 实现原图与特征结果的同步展示。
> *   **底层依赖攻坚**：通过环境变量注入与工作目录重定向，完美解决 x64 架构下非托管 C++ DLL 依赖崩溃的工业级难题。
> *   **异步与高并发**：摒弃阻塞式 UI，采用 `OnWorkEndStatusCallBack` 异步回调提取算子结果，结合 `Dispatcher.Invoke` 实现状态灯毫秒级无卡顿刷新。
> *   **软硬件全链路闭环**：封装 `SerialPort` 通信协议，根据视觉推理结果实时向外部设备下发控制指令。
